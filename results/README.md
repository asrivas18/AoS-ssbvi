# Results folder

This directory stores generated outputs from the simulation and figure scripts.

Typical contents include:

- Raw Monte Carlo output files in `.csv` format
- Summarized coverage tables and projection-dominance summaries
- Manuscript-ready figures in `.pdf`
- Preview figures in `.png`
- Session information for reproducibility

## Expected outputs

Depending on which scripts are run, this folder may contain files such as:

- `fig_coverage_vs_df_raw.csv`
- `fig_coverage_vs_df_summary.csv`
- `fig_coverage_vs_df.pdf`
- `fig_coverage_vs_df.png`

- `fig_coverage_vs_df_theta4_raw.csv`
- `fig_coverage_vs_df_theta4_summary.csv`
- `fig_coverage_vs_df_theta4.pdf`
- `fig_coverage_vs_df_theta4.png`

- `fig_projection_dominance_raw.csv`
- `fig_projection_dominance_summary.csv`
- `fig_projection_dominance.pdf`
- `fig_projection_dominance.png`

- outputs generated by `sim_sigma2_grid.R`
- outputs generated by `sim_theta4_grid.R`

- `sessionInfo.txt`

## Notes

- Large simulation outputs may be excluded from version control depending on the repository `.gitignore` settings.
- If you want to archive manuscript-ready outputs in GitHub, adjust `.gitignore` accordingly.
- For exact reproduction of paper results, rerun the scripts using the same Monte Carlo and bootstrap settings described in the manuscript and supplement.
